# Go Lang Playground

The purpose of this playground is to provide real working code.

## Description

Hear you will find some of the projects done in Go Programming Lang.

- Buf Manager
- GO REST APIs
- GRPC Example
- GRPC Error Handling
- HTTP Client and Server
- HTTPS Client and Server
- JSON Encoding

- Logging
- MongoDB Tutorial
- JSON Encoding
- Swagger Doc Gen
- Viper
- more...
